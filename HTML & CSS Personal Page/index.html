My Projects.

Flight Club Program

A program which allows users to sign up to recieve emails notifying them of good flight deals. The program uses a Google Sheet to keep a list of destinations to check and the historical low price for each of those locations. Those historical low prices are fed into a flight search API and when it finds a cheaper price, it will send the date and price to the user using the SMTP module in Python.

Stock Alert SMS App

A stock news monitoring program. The app monitor one particular stock (eg. Tesla) and compares the pricae at market close yesterday to market close the day before. If the price fluctuates 5% or more, the user is notified of the change and sent 3 news articles about Tesla whcih may give them insight into why the change occured. The program uses the AlphaVantage API to monitor the stock price and the NewsAPI to get the news articles.

GUI Quiz App

A Quizzler App made using Tkinter and which uses the Trivia API from the Open Trivia Database. The program selects 10 random trivia questions from the Open Trivia Database and displays them to the user. The user must answer either True or False by clicking either the green button or the red button. If the user answers correctly, the screen flashes green to indicate this to the user. If not, it will flash red. The user's score is also displayed on screen.

Password Manager App

A GUI password manager app made using Tkinter. The user is able to enter the name of a website and their login details for that site which is their username/email and a password which they can either create themselves or generate one randomly by clicking the "Generate Password" button. This information is stored locally in a CSV file. The user is also able to search for their login details by entering the name of the webiste.
